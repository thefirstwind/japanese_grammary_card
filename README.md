# 日语高级语法记忆卡片

打开 `index.html` 即可使用。系统已从 `n1n2语法_复习版_markdown_语音` 自动提取 292 张语法卡片。

## 模式

- 卡片记忆模式：默认显示语法答案、接续、用法和例句。阅读后可标记为“未完成 / 不熟悉 / 已掌握”。
- 测试模块：按当前筛选范围生成选择题，支持“看语法选中文意思”、“看语法选日文意思”、“看中文意思选语法”，以及第 12 章敬语专项的“看原型选敬语 / 看敬语选原型”。普通测试会排除第 12 章的 31 和 35，答对自动计入“已掌握”，答错自动计入“不熟悉”。

## 进度记录

进度会自动保存在当前浏览器的 localStorage 中，卡片记忆模式也会记住上次学习到的卡片。建议每天结束点击“导出进度”，得到 JSON 文件；换浏览器或设备时可以用“导入进度”恢复掌握进度。

## 2 天冲刺安排

- 第 1 天：完成前半部分 146 张。目标不是慢慢精读，而是把全部卡片过一遍，并把不会的标为“不熟悉”。
- 第 2 天：完成后半部分 146 张，然后筛选“未完成”和“不熟悉”进入测试模块，循环到统计区显示已掌握 100%。

推荐节奏：每 25 分钟 25 到 35 张，休息 5 分钟；每 2 到 3 轮后只刷“不熟悉”。
